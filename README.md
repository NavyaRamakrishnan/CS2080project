# CS2080project
## Overview
This repository is split into **backend** (Flask API) and **frontend** (React/Tailwind) folders.

### Backend
- Navigate to the backend folder:
  ```bash
  cd backend
  pip install -r requirements.txt
  python server.py
  ```

### Frontend
- Navigate to the frontend folder:
  ```bash
  cd frontend
  npm install
  npm start
  ```
